HTML 태그 [3]

Table

table - 표
tr - table row
td - table data

화면에 출력될 모든 내용은 td에만
th는 가운데 정렬에 진하게

<table border="1">
  <tr>
      <th>글번호</th>
      <th>글제목</th>
      <th>글쓴이</th>
  </tr>
  
  <tr align="center">
      <th>글번호</th>
      <th>글제목</th>
      <th>글쓴이</th>
  </tr>
  
  <tr bgcolor="red">
      <td>1</td>
      <td>노인과 바다</td>
      <td>헤밍웨이</td>
  </tr>
  
  <tr>
      <td bgcolor="blue">2</td>
      <td>오만과 편견</td>
      <td>제인 오스턴</td>
  </tr>
</table>




예제1

  • rowspan : 세로로 합치기
색깔 하얀색
노란색
연두색 
<table border="1">
  <tr>
      <td rowspan="3">색깔</td>
      <td>하얀색</td>
  </tr>
  
  <tr>
      <td>노란색</td>
  </tr>
  
  <tr>
      <td>연두색</td>
  </tr>
</table>




예제2

  • colspan : 가로로 합치기
색깔
하얀색 노란색 연두색 
<table border="1">
  <tr>
      <td colspan="3">색깔</td>
  </tr>
  
  <tr>
      <td>하얀색</td>
      <td>노란색</td>
      <td>연두색</td>
  </tr>
</table>
